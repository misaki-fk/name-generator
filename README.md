# 名前を授ける

あなたに"もう一つの名前"を授けるアプリです。
ペンネーム・ラジオネーム・ユーザーネームを、簡単に生成できます。

---

## 🪶 アプリ概要

好きな言葉や特徴を入力すると、
モードに応じた名前を自動生成します。

「生成する」のではなく、
**"名前を授ける体験"**をテーマにしています。

---

## ✨ 主な機能

* キーワード入力（最大3つ）
* モード選択

  * かわいい
  * 厨二
  * ギャル
  * ビジネス
  * アイドルオタク
* 名前の自動生成
* 名前の理由表示

## 🚀 起動方法（ローカル）

### Dockerを使用する場合

```bash
docker compose up
```

ブラウザで以下にアクセス：

```
http://localhost:5173
```

---

### Dockerを使わない場合

```bash
npm install
npm run dev
```

## 🎯 制作背景

ペンネームやラジオネームを考えるとき、
「しっくりくる名前が思いつかない」と感じたことから制作しました。

単なるランダム生成ではなく、
**"今の自分に合う名前を授ける"体験**を目指しています。
