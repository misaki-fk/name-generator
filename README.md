# アプリ名
「名前を授ける」

## アプリ概要
好きな言葉や特徴を入力すると、
モードに応じた名前を自動生成します。

ペンネームやラジオネームを考えるとき、
「しっくりくる名前が思いつかない」と感じたことから制作しました。


## このアプリの開発目的
* React TypeScriptのキャッチアップ
* Claude Codeの使用、ハーネスを意識しコードを全て出してもらうのではなく、学習理解の補助として活用する


## 主な機能
* キーワード入力（最大3つ）
* モード選択
  * かわいい
  * 厨二
  * ギャル
  * ビジネス
  * アイドルオタク
* 名前の自動生成
* 名前の理由表示
* Xでのシェア機能

## 起動方法（ローカル）
### Dockerを使用

```bash
docker compose up
```

ブラウザで以下にアクセス：

```
http://localhost:5173
```



### Dockerを使わない場合

```bash
npm install
npm run dev
```
